Real Estate Shopping Website

Project information

  • Category: Web Application
  • Technology: PHP, HTML, CSS
  • Project URL: https://github.com/mahidharseelam/royaldevelopers

Developed a web application where a user can buy and sell a property. The user interface developed using HTML and CSS, backend using PHP with the help XAMPP server. Used MySQL database for store and fetch data. Team of 3 and developed using Visual Studio Code. I was responsible in overall structure and functionality development of the application.
